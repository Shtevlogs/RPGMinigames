# Wild Mage: Pre-Drawn Card Tracking

**Priority**: 12 (Chunk 2: Combat & Minigames)

## Description
Implement pre-drawn card system. Basic attacks "pre-draw" a card to be used in the next minigame (doesn't count toward hand size).

## Requirements
- Store pre-drawn card from basic attacks
- Include pre-drawn card in next minigame hand
- Pre-drawn card doesn't count toward hand size limit
- Reroll pre-drawn card on subsequent basic attacks
- Visual indicator for pre-drawn card
