# Time Wizard: Event Effects System

**Priority**: 09 (Chunk 2: Combat & Minigames)

## Description
Implement timeline event effects system. Different event symbols (Square, Pentagon, Hexagon) trigger different effects based on equipment.

## Requirements
- Trigger effects based on event symbol activated
- Scale effect power with board completion percentage
- Equipment determines which effects each symbol provides
- Effects include: turn manipulation, party buffs, temporal damage, Speed buffs/debuffs
- Grid position may influence targeting
