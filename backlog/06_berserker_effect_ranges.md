# Berserker: Effect Ranges System

**Priority**: 06 (Chunk 2: Combat & Minigames)

## Description
Implement effect ranges system for Berserker blackjack minigame. Basic attacks add effect ranges that trigger special effects when standing on scores within the range.

## Requirements
- Generate effect ranges from basic attacks (when not berserking)
- Strategy attribute determines number of random effect ranges
- Effect ranges cleared after ability use
- Trigger effects when standing on scores within range
- Equipment can add additional effect ranges
- Examples: 6-8 = party Luck +1, 18-20 = taunt enemy
